# Golang代码案例

这是很久之前我在学习Golang中挨个敲过的一部分代码案例,基本上如果在看Golang的书时敲了书上的代码,再敲这些案例之后就差不多入门了.

-   一般的代码学习自 **go by example**
-   加密部分的代码主要是通过文档和之前的编程经验结合学习自黑马程序员的公开课程 **4天掌握GO语言密码学**
-   还有一部分是来自一个APP **Code Recipes Pro**
-   还有一部分是工作中遇到的问题
